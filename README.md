# fonte-chaveada-buck
Desenvolvimento de fonte chaveada topologia Buck controlada por PWM (SG3525)

Tensão de entrada (VS): 120Vdc (Vs_min) a 325Vdc (Vs_max) (o que corresponde a 85Vrms e 230Vrms da rede elétrica convencional)

Tensão de saída (V_o): 48Vdc

Corrente de saída (I_omax): 600mA

Frequência de chaveamento (f): 80kHz

Potência de saída (P_out): 28,8W

Percentual de ripple (〖_ripple〗): +/-1%

Components principais:

SG3525

IR2110

IRF840

Os arquivos em anexo podem ser baixado o esquematico utilizado e os calculos para o programa Octave.
